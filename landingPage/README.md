# SK-care Landing Page

A modern, responsive landing page for **SK-care**, featuring high-performance skincare products. This project highlights the *LA ROCHE-POSAY ANTHELIOS UVAIR SERUM SUNSCREEN SPF50+*.

## 🌟 Features

- **Responsive Design**: Optimized for various screen sizes using Tailwind CSS.
- **Product Showcase**: Detailed information about the Anthelios UVAIR Serum Sunscreen.
- **Key Benefits Section**: Clear bullet points highlighting product advantages like 16-hour protection and 24-hour hydration.
- **Interactive Navigation**: Easy access to Product Types, Contact, and Skin Test sections.

## 🛠️ Tech Stack

- **HTML5**: Structural foundation.
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development (via CDN).
- **Custom CSS**: Additional styling in `stlye.css`.
